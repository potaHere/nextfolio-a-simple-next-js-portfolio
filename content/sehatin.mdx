## Description
A mobile application designed to help users make healthier food choices by identifying fruits and vegetables through photos. It provides detailed health information, personalized meal suggestions, and nutritious recipes. The app aims to address health issues like hypertension and stunting in Indonesia by promoting informed dietary habits among its users.  

## Role
Backend

- Setup evirotment on GCP 
- deploying backend on GCP using App Engine
- Create API for calling vegetables & fruits information
- Create API for data on Jember's fruit and vegetable shops
