## Description
The website-based application uses PHP language for Rembangan Tourism which makes it easier for
tourists or visitors to book tickets and facilitates management with data visualization dashboards to
monitor and make future strategies.  

## Role
Full Stack

- Create Landing Page 
- Create login and ticket page
- Setup payment gateway using Midtrans
