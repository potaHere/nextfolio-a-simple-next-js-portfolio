## Description
MedReyGen is a web-based AI assistant for classifying respiratory diseases (pneumonia, tuberculosis, and COVID-19) from chest X-rays using TensorFlow. It also features a Vertex AI-powered chatbot that provides medical info based on classification results. Built for accessibility and speed, MedReyGen supports faster medical decision-making in underserved areas.
